pynmealogger
==============


Install
-------

Developer
_________

Install as a user

.. code:: bash
	  
   python setup.py develop --user

Uninstall as a user
   
.. code:: bash
	  
   pip uninstall pynmealogger


Dependencies
------------

pynmeatool does need pynmea2:

* `pynmea2 <https://pypi.python.org/pypi/pynmea2>`_
